About Ghostfolio

Ghostfolio ({{ version }}) is open source software which empowers busy folks to have a sharp look of their financial assets and to make solid, data-driven investment decisions by evaluating automated static portfolio analysis rules. The project has been initiated by Thomas Kaul. This instance has been last published on {{ lastPublish }}.

If you encounter a bug or would like to suggest an improvement or a new feature, please open an issue at GitHub, tweet to @ghostfolio_ or send an e-mail to hi@ghostfol.io.
